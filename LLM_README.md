## Purpose
This repository stores all data models, datasets, Power BI files, ESG metrics, and operational dashboards used across Trusted Treks.

This repo is the intelligence layer for:
- Dispatch performance insights  
- Customer volume + demand forecasting  
- Driver scorecards and performance metrics  
- EV efficiency + charging analysis  
- Sustainability/ESG reports  
- Financial KPIs  
- Corporate reporting  

It centralizes all data used for analysis, visualization, and automated decision-making.

## What AI May Do
- Document dataset schemas  
- Describe Power BI dashboard usage  
- Generate DAX formulas (on request)  
- Create metric definitions (KPIs, scorecards)  
- Provide forecasting methodologies  
- Generate instructions for importing datasets  
- Draft new dashboard pages (concept only)  

---

## What AI May NOT Do
- Upload or modify actual customer data  
- Change existing PBIX files  
- Generate fake data unless requested explicitly  
- Move or rename dataset folders  
- Delete or overwrite dataset structures
